# ROADSOS

Monorepo with:
- Expo mobile app in `apps/mobile`
- Firebase backend in `firebase`

## Quick start

1. Install deps once at the repo root:
   - `npm install`
2. Start the mobile app:
   - `npm run mobile`
3. Start Firebase emulators:
   - `npm run emulators`

## Notes

- The HTML UI files in the workspace root are design references only.
- Add Firebase and API keys to `apps/mobile/.env` (see `apps/mobile/.env.example`).
