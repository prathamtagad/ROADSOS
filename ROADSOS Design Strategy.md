# ROADSOS Design Strategy

## Overview
ROADSOS is a mission-critical emergency tool for the BIMSTEC region. The design prioritizes speed of action, offline reliability, and cross-border intelligence.

## Core UX Principles
1. **Zero-Latency SOS**: The primary red button must be accessible within 1 tap from the home screen.
2. **High Visibility**: High contrast for outdoor use (bright sun or dark roads).
3. **Status Clarity**: Users must know exactly which country's contacts are active (Thailand, Myanmar, etc.).
4. **Information Density**: Emergency cards must show life-saving info (Blood Type, Allergies) without scrolling.

## Visual Identity
- **Red**: Emergency / SOS / Immediate Danger.
- **Yellow**: Breakdown / Road Assistance / Warning.
- **Green**: Verified data / Safe status.
- **Blue**: Information / Embassies / Insurance.
